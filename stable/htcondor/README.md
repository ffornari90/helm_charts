## Deployment on Kubernetes with Helm

### Quick Start

```bash
helm repo add dodas https://dodas-ts.github.io/helm_charts
helm repo update
helm install htcondor dodas/htcondor
```

> **Tip**: You can use the default [values.yaml](values.yaml)
